# TooliVerse AI Workplace Memory & Productivity Hub

## 🚀 Project Overview

TooliVerse AI Workplace Memory & Productivity Hub is an AI-powered workplace productivity and knowledge management platform designed to help professionals automate repetitive tasks while preserving and retrieving valuable organizational knowledge.

Modern workplaces generate large amounts of information through meetings, emails, reports, research documents, and project discussions. Employees often spend significant time searching for information that already exists, leading to reduced productivity and knowledge loss.

TooliVerse addresses this challenge by combining AI-powered workplace automation with a Workplace Memory Engine that stores, organizes, and retrieves workplace knowledge on demand.

The platform acts as both:

* A Productivity Assistant that automates routine workplace tasks.
* A Workplace Memory System that captures and preserves organizational knowledge.

This project was developed as part of the AI Skills Accelerator Programme and demonstrates practical applications of AI, prompt engineering, productivity enhancement, and responsible AI practices.

## ⚙️ Setup Instructions

### Prerequisites

Ensure you have the following installed:

* Node.js (v18 or later)
* npm or yarn
* Git

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/tooliverse-ai-workplace-hub.git
```

### 2. Navigate to the Project Folder

```bash
cd tooliverse-ai-workplace-hub
```

### 3. Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

### 4. Configure Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_OPENAI_API_KEY=your_api_key_here
```

Add any additional API keys required by your implementation.

### 5. Start the Development Server

```bash
npm run dev
```

or

```bash
yarn dev
```

### 6. Open the Application

Open your browser and navigate to:

```text
http://localhost:5173
```
